## 3. Описание UI складовщика

### 3.1 Назначение
UI складовщика предназначен для сотрудников склада и администраторов кондитерской фабрики «Невский Кондитер». Интерфейс позволяет управлять заказами, отслеживать их статусы и просматривать статистику продаж.

### 3.2 Доступ
- Доступен только пользователям с ролью `admin`
- Кнопка «Склад / Заказы» отображается в разделе «Профиль» только для администраторов
- Демо-доступ: admin@nk.ru / admin1234

### 3.3 Структура экрана

```
┌─────────────────────────────────┐
│ ← Склад                    🔄  │  ← Хедер с кнопкой обновления
├─────────────────────────────────┤
│ ┌──────────┐ ┌──────────┐      │
│ │ Заказы   │ │ Выручка  │      │  ← Статистика (4 карточки)
│ │   12     │ │ 8 450 ₽  │      │
│ └──────────┘ └──────────┘      │
│ ┌──────────┐ ┌──────────┐      │
│ │ Юзеры    │ │ Товары   │      │
│ │    5     │ │   25     │      │
│ └──────────┘ └──────────┘      │
├─────────────────────────────────┤
│ Все(12) Новый(3) Сборка(2) ... │  ← Фильтры по статусу
├─────────────────────────────────┤
│ Заказы                          │
│ ┌───────────────────────────┐  │
│ │ NK-A1B2C3D4    [Новый]    │  │  ← Карточка заказа
│ │ user@mail.ru              │  │
│ │ So crispy клубника × 2    │  │
│ │ PROBATON кокос × 1        │  │
│ │ ─────────────────────     │  │
│ │ 2024-01-15      250 ₽    │  │
│ │ [  Изменить статус  ▾  ]  │  │  ← Кнопка смены статуса
│ └───────────────────────────┘  │
│ ┌───────────────────────────┐  │
│ │ NK-E5F6G7H8  [Доставлен]  │  │
│ │ ...                        │  │
│ └───────────────────────────┘  │
└─────────────────────────────────┘
```

### 3.4 Функции складовщика

| Функция | Описание |
|---------|----------|
| Просмотр статистики | 4 карточки: заказы, выручка, пользователи, товары |
| Фильтрация заказов | Чипы по статусам с количеством заказов в каждом |
| Просмотр заказа | Номер, email, список товаров с ценами, итого, дата |
| Смена статуса | Dropdown: Новый → Сборка → Отправлен → Доставлен / Отменён |
| Обновление данных | Кнопка 🔄 в хедере для повторной загрузки |

### 3.5 Жизненный цикл заказа

```
Новый → Сборка → Отправлен → Доставлен
  │                              
  └──────────→ Отменён           
```

| Статус | Цвет | Действие складовщика |
|--------|------|---------------------|
| Новый | 🔵 Синий | Принять в работу → «Сборка» |
| Сборка | 🟠 Оранжевый | Собрать заказ, передать курьеру → «Отправлен» |
| Отправлен | 🟣 Фиолетовый | Ожидание доставки → «Доставлен» |
| Доставлен | 🟢 Зелёный | Финальный статус |
| Отменён | 🔴 Красный | Финальный статус |

|--------|----------|----------|
| new | #5E5CE6 | NkBlue |
| processing | #FF9F0A | NkOrange |
| shipped | #8B7BFF | NkViolet |
| delivered | #34C759 | NkSystemGreen |
| cancelled | #FF375F | NkRed |
