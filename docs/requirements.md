# Невский Кондитер — ЗОЖ

**Версия:** 2.0  
**Дата:** 2026-06-02

## 2. Нефункциональные требования (NFR)

### 2.1 Производительность

| ID | Требование |
|----|-----------|
| NFR-01 | Время загрузки каталога продуктов — не более 2 секунд при стабильном соединении |
| NFR-02 | Время отклика API — не более 500 мс для основных эндпоинтов |
| NFR-03 | Плавная прокрутка каталога (60 fps) при отображении до 50 товаров в LazyVerticalGrid |
| NFR-04 | Ленивая загрузка и кэширование изображений через Coil |
| NFR-05 | Timeout на сетевые запросы: подключение 5 сек, чтение 10 сек |

### 2.2 Безопасность

| ID | Требование |
|----|-----------|
| NFR-06 | JWT-аутентификация для защищённых эндпоинтов |
| NFR-07 | Хранение JWT-токена в зашифрованном DataStore Preferences |
| NFR-08 | Пароли хешируются на сервере с использованием bcrypt (cost factor 10) |
| NFR-09 | JWT-секрет хранится в конфигурационном файле / переменной окружения, не в коде |
| NFR-10 | Проверка роли пользователя на стороне сервера для всех админ-эндпоинтов (middleware requireAdmin) |
| NFR-11 | Параметризованные SQL-запросы через Exposed ORM (защита от SQL-инъекций) |
| NFR-12 | CORS настроен на разрешённые домены |
| NFR-13 | Каскадное удаление связанных данных при удалении пользователя/товара/категории |
